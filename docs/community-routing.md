# Community Routing

Use Discussions for general evaluator-kit questions, documentation feedback, or broader conversation about the managed execution endpoint.

Use Issues when you have one structured payload to share or one short technical question that fits the repo templates.

Email [tetsutetsu11@icloud.com](mailto:tetsutetsu11@icloud.com) when you need to request an evaluation API key, send sensitive material, or handle timing constraints that should not sit in a public thread.

Use [submit-payload.html](../submit-payload.html) when the next step is one payload, one target schema, a stated downstream risk, and a clear return method through polling or webhook.
